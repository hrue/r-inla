\section*{Linkmodel: \texttt{tan} and \texttt{tanpi}}

These are link-models for the circular models. 

\subsection*{Parametrization}
The \texttt{tan} link is
\begin{displaymath}
    z = g^{-1}(x) =2\arctan(x)
\end{displaymath}
and
\begin{displaymath}
    x = g(z) = \tan(z/2)
\end{displaymath}
and $z\in(-\pi,\pi)$

The \texttt{tanpi} link is
\begin{displaymath}
    z = g^{-1}(x) =2\arctan(x) + \pi
\end{displaymath}
and
\begin{displaymath}
    x = g(z) = \tan((z-\pi)/2)
\end{displaymath}
and $z\in(0,2\pi)$
