Abstract

This tutorial will show you how to fit models that contains at least one effect specified from an SPDE using the ‘R-INLA‘. We will use an SPDE based model here to define random effects over continuous one- or two- dimensional domains. The usual application is data whose geographical location is explicitly considered in the analysis. This tutorial explores ’R-INLA’ functionalities by using examples. It starts with simple models and increases in complexity.

In Chapter 1 we will introduce the Gaussian random fields and the SPDE framework, work with a toy example and work through some examples of building a mesh. Then we consider three detailed examples in Chapter 2. They address non-Gaussian data, survival analysis and inclusion of covariates in the covariance parameters. In Chapter 3 we copy random fields to model two or more outcomes jointly. It includes a measurement error model, a coregionalization model and considering part or the entire linear predictor from one outcome in a linear predictor to another one. The log Cox point process model is considered in Chapter 4. Finally, Chapter 5 presents example analysis of spacetime data.

To get started you **must first read** the tutorial at http://www.r-_inla.org/examples/tutorials/spde-_tutorial-_from-_jss. If you are in a rush to fit a simple geostatistical model, please see http://www.math.ntnu.no/inla/r-_inla.org/tutorials/spde/inla-_spde-_howto.pdf. Or you can type vignette(SPDEhowto) for a two dimensional example or vignette(SPDE1d) for a one dimensional example. To understand mesh building you may find it helpful to play with the mesh building demonstration in Shiny which you can open by typing demo(mesh2d).
